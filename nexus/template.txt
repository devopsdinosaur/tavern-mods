[size=4][b][u]Installation[/u][/b][/size]

[list=1]
[*][url=https://github.com/LavaGang/MelonLoader/releases/tag/v0.7.1]Install MelonLoader[/url]
[*]Run the game and quit to allow MelonLoader to create assemblies and Mods dir.
[*]Download and extract this mod.
[*]Copy the extracted files to [i]<[[game]]-home>/Mods[/i] folder.
[*]Start the game again and enjoy!
[/list]

[size=4][b][u]Config Options[/u][/b][/size]

Note that all of the options in this mod default to off or game defaults (false, zero, blank, etc) to enable the user to have as little of a "cheaty" experience as possible. 

For information on how to modify these options, see the [i]Editing the Config File[/i] section below.

[[config_options]]
[size=4][b][u]Editing the Config File[/u][/b][/size]

[list=1]
[*]Open notepad or your favorite text editor (Windows Key + R then 'notepad').
[*]File => Open.  Navigate to [i]<[[game]]-home>/UserData[/i] and open [i]MelonPreferences.cfg[/i].
[*]Note that this file stores config data for all MelonLoader mods for this game.  Sections/categories are denoted by strings surrounded by brackets [].  Search the file for '[[underscore_guid]]' and modify the values under those sections only.  Lines starting with '#' are just comments.
[*]After making changes, save the file and restart the game to try out the new settings!
[/list]
